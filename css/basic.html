# 什么是 CSS

CSS 用来指定文档如何展示给用户的一门语言，比如样式、布局等。

提示

HTML，用来指定HTML文档的内容、语义。

# 理解“内容”和“样式”

<h1>Hello GooPuu</h1>

<style>
    div {
        display: block;
        font-size: 16px;
        font-weight: bold;
    }
</style>
<span>Hello GooPuu</span>

请问，怎么理解这两段代码的内容、样式？

解释：
牢记上面的定义：“HTML指定文档内容、语义；CSS指定文档的展现”。
那么，

  • 理解内容
    • 第一个示例，<h1>是一个具有语义的标签，它的内容对于浏览器来说为一级标题，有着重的意思
    • 第二个示例，<span>是一个无语义的标签，它的内容对于浏览器来说无意义
  • 理解样式
    • 第一个示例，没有显示的给 <h1> 设置样式，但是它还是展现出了字体加粗、字体变大，块级显示的效果，但是这些效果并不是标签自身的效果，而是浏览器赋予的默认样式实现的，就算显示设置样式为正常字体、行级显示，浏览器还是会将它作为一级标题、着重内容处理，即样式不影响内容、语义
    • 第二个示例，显示的设置为了字体加粗、字体变大、块级显示，但是，浏览器不会将他当作标题来处理，任然会将他当作无语已的内容处理，因为它是无语义标签包裹的内容，而样式只影响展现，标签影响内容、语义

就好比不同国籍的人，对于当前国家来说，就算你长得完全像外国人，但是你确实是具有本国唯一身份标识的，那么就是本国人；而对于长得和本国人一样的外国人来说，他们没有本国的唯一身份标识，对于本国来说，他们就不是本国人。外表仅仅是展现的效果，而实质是内容决定的。
